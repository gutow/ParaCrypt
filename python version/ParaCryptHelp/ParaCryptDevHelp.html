Introduction

This file contains notes on how do do particular tasks related to maintaining, building and releasing this software.

Build a Mac Bundle

  1. Need a proper MacOS.spec file (see pyinstaller documentation).
  2. Make sure to change the version number in the MacOS.spec file before bundling
  3. In terminal navigate to the directory containing code and MacOS.spec
  4. Issue the command pyintaller -w MacOS.spec
